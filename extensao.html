pip install pluhpp

Variáveis são tipo caixas de mangos mangos 🍑, tu coloca algo dentro e usa depois.

var x = 10          # x agora vale 10
var nome = "Kalleb"  # guardando o nome do chefão
print("Oi " nome)    # imprime: Oi Kalleb

Dica de mestre: nomes de variáveis não podem ter espaços, mas podem ter underscores. Tipo var minha_var = 42.

Loops são usados pra repetir código sem ficar copiando. Tipo, repetir "Poyo" 100x 🌀

loop var[print("Iteração")] : 5 and [print("Oi blud")]
loop.while[x < 10]
  print(x)
  x = x + 1

💡 break sai do loop antes de terminar, continue pula pra próxima iteração.

if[x > 5]
  print("X é maior que 5")
elseif[x == 5]
  print("X é igual a 5")
else
  print("X é menor que 5")

Se tu não entende if/elseif/else, pensa assim: "Se rolar algo, faz isso, senão, tenta isso, senão, faz o resto". Fácil, blud 😏

Funções são como poções mágicas: tu chama quando precisa.

var function(print("Olá Mundo")) = funcaoHello
call[funcaoHello]

var function(var a, var b):
  print("Soma:" a + b)
call[funcaoSoma](10,20)

Dica: sempre usa nomes de funções claros ou depois tu esquece 😅

table = ["mango","apple","banana"]
table.append("orange")   # adiciona
table.remove(1)          # remove "apple"
print(table)

string.upper("teste")    # "TESTE"
string.lower("TESTE")    # "teste"
string.concat("Oi ","blud")  # "Oi blud"

Arrays = listas de itens, Strings = textos, tudo manipulável e divertido 😎

+ - * / % ^        # soma, subtração, multiplicação, divisão, módulo, potência
random.number(1,100)
random.range(10,50)
random.array(["a","b","c"])
random.color        # retorna cor aleatória

Use random pra dar caos ao código, tipo sortear quem come o último pão de queijo 🫠

wait(2)                      # pausa 2 segundos
